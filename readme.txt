How to run this program?
1. Clone this program from https://github.com/vvlnote/romanNumeralConverter.git
2. in the local machine, node.js needed to be installed.
3. open the terminal, and go to where this program is located in the local machine
4. type 
		node romanNumberConverter.js
	then there is a default number will be converted into Roman numerals
5. type 
		node romanNumberConverter.js 891
	then 891 will be read in as input number, and screen will print out the corresponded Roman numerals
6. only accept one input number every time when runs this program in the command line
